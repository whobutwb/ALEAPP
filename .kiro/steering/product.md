# ALEAPP - Android Logs Events And Protobuf Parser

ALEAPP is a digital forensics tool for parsing and analyzing Android device extractions. It processes various data sources (filesystem, tar, zip, gz archives) and generates comprehensive HTML/TSV reports with timeline analysis.

## Purpose
- Parse Android full system extractions for forensic analysis
- Extract artifacts from apps, system logs, databases, and protobuf files
- Generate structured reports (HTML, TSV, KML) for investigators
- Support timeline reconstruction across multiple data sources

## Key Features
- Plugin-based architecture for artifact parsing (200+ artifact modules)
- CLI and GUI interfaces
- Profile system for selective artifact processing
- Case data management for forensic documentation
- Cross-platform support (Windows, macOS, Linux)

## Target Users
Digital forensics investigators and analysts examining Android device data.
